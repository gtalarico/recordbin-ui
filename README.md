# RecordBin Ui

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

## Deployment

```
heroku login
heroku app create <appname>
heroku git:remote --app <appname>
git push heroku
```
